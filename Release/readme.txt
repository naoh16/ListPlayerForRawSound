■   I. はじめに________________________
    raw ファイル(ヘッダー無しwaveファイル)をリスト表示して連続再生す
  るためのツールです。必ずしも音声でなくても再生していまいます。大量
  の連続したファイルを聞いてより分ける作業に最適です。


■  II. フォルダ構成____________________
  ./
    +- ListPlayer.exe       実行ファイル
    +- readme.txt           このファイル


■ III. 開発・動作環境__________________
  □ 開発環境
       Windows 10
       Delphi 10.3 Community Edition

  □ 動作確認
       Windows 10


■  VI. インストール・アンインストール__
    実行ファイルを適当なフォルダに入れてください．
  初回起動時に、ListPlayer.iniを作成し、そこに設定が保存されます。
    アンインストールは実行ファイル及び設定ファイル(ListPlayer.ini)
  を消去してください。レジストリはいじっていません。


■   V. 使用方法________________________
    ListPlayer.exe を起動する．または、ファイルもしくはフォルダを
  ListPlayer.exeにドラッグアンドドロップすると、そのファイルのある
  フォルダから開きます。
    または、次の手順でファイルを選択していきます。
    
    １．左上のコンボボックスでドライブを選択
    ２．左のツリーからフォルダを選択
    ３．右に表示されるリストからファイルを選択
    
  ダブルクリックまたはリターンキーで再生を行います。
  また、[Play]ボタンを押すとそのリスト内のファイルを連続して再生しま
  す。（わずかにファイル間にポーズがはいります）
  スペースを押すと停止もしくは次の曲を再生します。

  ファイルリスト関連では以下のショートカットが有効です。
    Alt + UP           ・・・ ドライブ コンボボックス
    Alt + LEFT or DOWN ・・・ ディレクトリ ツリー
    Alt + RIGHT        ・・・ ファイル リスト

  他、操作については、[ヘルプ(H)] -> [簡易ヘルプ表示] を見てください。


■  VI. 外部ツールについて______________
    主にspwaveと連携することを考えて作ってます。
  設定ダイアログの引数には以下の変数が使えます。
  □ %b
       現在設定しているビット数に置換されます。

  □ %f
       現在設定しているサンプリング周波数に置換されます。

  □ %c
       チャンネル数に置換されます。Stereoチェックボックスがonの時、
     '2'、offの時、'1'と置換されます。

  □ %t
       ファイルフォーマット文字列に置換されます。Swapチェックボックス
     がonの時、'swap'、offの時、'raw'という文字列に置換されます。
   
   おすすめ設定例：
     名前：spwave
     場所：(spwave.exeの場所を指定してください)
     引数：-f %f -b %b -c %c -format %t
